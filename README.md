| Duration (Days) | Interest (Profit) |
| :---            | :---              |
| **7 Days**      | 3%                |

### B. Direct Income
Referral commission un logon ke liye jo naye users ko system mein laate hain.
* **Rate:** 3%
* **Condition:** ID kam se kam **$20** se activate honi chahiye.
* **Note:** Re-topup par Direct Income nahi milti.

## 4. Terms & Conditions
System ke rules aur regulations:
1. **Re-Topup Rule:** Saari incomes continue rakhne ke liye, ID ko same ya higher package se re-topup karna zaroori hai.
2. **Withdrawal Limit:** Minimum withdrawal amount **$5** hai.
3. **Income Release:** Working income daily release hoti hai (minimum $5 hone par).
4. **Admin Control:** System fully decentralized hai, koi manual admin control nahi hai.
